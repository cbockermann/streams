\subsubsection{\label{api:stream:io.ArffStream}Class {\ttfamily stream.io.ArffStream}}

This stream implementation provides access to reading ARFF files and
processing them in a stream based fashion. ARFF is a standard format
for data in the machine learning community which has its root in the
WEKA project \cite{weka}.
