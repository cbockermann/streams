\Processor{SetValue}

This processors allows for setting a feature to a single, constant
value:
\begin{figure}[h!]
\centering
\begin{lstlisting}{lang=xml}
  <SetValue key="attribute1" value="abc" />
\end{lstlisting}
\end{figure}
