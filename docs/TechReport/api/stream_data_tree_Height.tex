\paragraph{Height}

This processor computes the height of trees implementing the
\texttt{TreeNode} interface. The height is computed for all attributes
of type TreeNode or the specified keys, if the \texttt{keys} parameter
has been set.

The height of each tree is stored as attribute \texttt{height(...)}. For
example, in the following snippet, the Height processor will create a
new attribute \texttt{height(@sql:tree)} with the height of the tree
stored in \texttt{@sql:tree}:

\begin{verbatim}
  &lt;Height keys="@sql:tree" /&gt;
\end{verbatim}
