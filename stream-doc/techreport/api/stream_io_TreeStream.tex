\subsubsection{TreeStream}

This stream is a line oriented stream that expects each line to provide
a tree structure in the format

\begin{verbatim}
  ( ROOT ( A1 ( A1.1 ) ( A1.2 ) ) ( A2 ) )
\end{verbatim}
